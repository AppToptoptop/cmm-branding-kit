# Pika — Test Guide

## Use Cases CMM

1. **Clips viraux** — contenu meme-able
2. **Réactions** — expressions faciales exagérées
3. **Quick cuts** — éléments pour montage TikTok

## Test 1 : Phone Vibration Chaos

### Prompt

```
A phone on a table vibrating intensely with an incoming call. Objects around it shake. Comedic exaggeration. The caller ID shows "YOUR COACH". Fast-paced, energetic, meme aesthetic.
```

### Paramètres

| Paramètre | Valeur |
|-----------|--------|
| Duration | 3 seconds |
| Style | Fun/exaggerated |

## Test 2 : Notification Ignore Montage

### Prompt

```
A thumb repeatedly swiping away phone notifications. Each swipe more aggressive. Notifications pile up faster and faster. Frustration building. Comedic timing. Ends with phone thrown aside.
```

## Test 3 : Answering The Call (Dramatic)

### Prompt

## Test 4 : Scale Celebration

### Prompt

```
A person stepping on a bathroom scale. Their face transforms from nervous to ecstatic. Confetti explodes (implied). Victory dance. Comedic joy. Phone visible in their other hand showing a call.
```

## Pika Features uniques

- **Modify region** : Animer une partie spécifique d'une image
- **Expand canvas** : Étendre les bords d'une vidéo
- **Lip sync** : Ajouter du lip sync à une image
- **Sound effects** : Génération auto de son
