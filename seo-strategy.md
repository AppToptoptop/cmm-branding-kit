## 4. Technical SEO

### 4.1 Site Speed Recommendations

| Action | Priority | Impact |
|---|---|---|
| Use Next.js or Astro with SSG for blog | P0 | Critical for Core Web Vitals |
| Optimize images: WebP format, lazy loading, srcset | P0 | LCP improvement |
| Minimize JavaScript bundle (< 100KB initial) | P0 | FID / INP improvement |
| Use CDN (Vercel Edge, Cloudflare) | P0 | TTFB < 200ms |
| Preload critical fonts (Sora, DM Sans) | P1 | CLS reduction |
| Inline critical CSS, defer non-critical | P1 | FCP improvement |
| Enable Brotli compression | P1 | Transfer size reduction |
| Set cache headers: immutable for static assets | P1 | Repeat visit speed |

**Target Core Web Vitals:**
- LCP (Largest Contentful Paint): < 2.5s
- INP (Interaction to Next Paint): < 200ms
- CLS (Cumulative Layout Shift): < 0.1

### 4.2 Mobile Optimization

| Requirement | Implementation |
|---|---|
| Mobile-first responsive design | CSS media queries, fluid typography |
| Touch targets minimum 48x48px | All buttons, links, CTAs |
| No horizontal scroll | max-width: 100%, overflow-x: hidden |
| Sticky CTA on mobile | Fixed bottom bar with "Essai Gratuit" button |
| Font sizes: minimum 16px body | Prevent iOS zoom on input focus |
| Viewport meta tag | `<meta name="viewport" content="width=device-width, initial-scale=1">` |
| AMP consideration | Not recommended -- focus on fast PWA instead |

## 7. Partner SEO Synergy

### 7.1 How the Partner Perks Program Creates Natural Backlinks

The partner program is designed to be an SEO flywheel:

```
CMM creates partner page with dofollow links
       |
       v
Partners link back to CMM from their site
       |
       v
Google sees natural link exchange between relevant domains
       |
       v
Both CMM and partner gain domain authority
       |
       v
Higher rankings attract more partners
       |
       v
More partners = more backlinks = higher DA
       |
       (cycle repeats)
```

**Natural Link Contexts:**
- Partner blog posts: "Nous sommes partenaire de Call Me Maybe, le coach IA..."
- Partner product pages: "Disponible pour les utilisatrices Call Me Maybe"
- Partner email newsletters: Link to co-branded content
- Partner social media: Links to shared landing pages
- Press releases: Both brands mentioned with links

### 7.2 SEO Value Proposition for Partners

When recruiting partners, lead with SEO benefits:

**Pitch to partners:**

> "En devenant partenaire Call Me Maybe, vous beneficiez de :
> 1. **Backlink dofollow** depuis callmemaybe.ai (DA en croissance)
> 2. **Exposition a notre audience** de femmes 25-40 interessees par la sante
> 3. **Co-creation de contenu** : articles de blog, comparatifs, guides
> 4. **Mentions dans notre newsletter** avec lien vers votre site
> 5. **Badge partenaire** a afficher sur votre site (avec lien retour)

> En echange, nous vous demandons un lien dofollow depuis votre site et une mention dans vos communications."
